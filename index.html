📖 O que é Regra de Três?

A Regra de Três é um método matemático para resolver problemas envolvendo grandezas proporcionais. É uma das ferramentas mais úteis da matemática básica.

Se a ↔ b
e c ↔ x
Então: x = (b × c) / a
Exemplo Prático:
Se 2 maçãs custam R$ 4,00, quanto custam 5 maçãs?

2 maçãs → R$ 4,00
5 maçãs → x

x = (4 × 5) / 2 = R$ 10,00

🔄 Proporção Direta vs Inversa

Direta: Quando uma grandeza aumenta, a outra também aumenta (ex: mais produtos = mais preço).

Inversa: Quando uma grandeza aumenta, a outra diminui (ex: mais trabalhadores = menos tempo).

Direta: x = (b × c) / a

Inversa: x = (a × b) / c

📖 O que é Notação Científica?

A Notação Científica é uma forma de escrever números muito grandes ou muito pequenos de maneira compacta, usando potências de base 10.

N = a × 10ⁿ
onde: 1 ≤ |a| < 10 e n ∈ ℤ
Exemplos:
300.000.000 = 3 × 10⁸ (velocidade da luz em m/s)
0,000000001 = 1 × 10⁻⁹ (1 nanômetro em metros)
6.022.000.000.000.000.000.000.000 = 6,022 × 10²³ (Número de Avogadro)

➕ Operações
